# GBA Yu-Gi-Oh! booster packs

In-game booster/card packs for the GBA games that have them, one file per
pack: a `# metadata` header then card names grouped by `## Rarity`. Card names
are resolved to the active v6.0 pool spelling; out-of-pool cards are kept and
flagged (`_MISMATCHES.md`). **Filter/random packs** (e.g. "All Effect Monsters",
"Dorothy's Gift") have no fixed list — their files record the rule instead.

Source: Yu-Gi-Oh! Fandom wiki (`(XXX-BP)` set pages), via the MediaWiki API.
**The Sacred Cards** and **Reshef of Destruction** are RPGs with no packs (cards are won through the story), so they have no folder here.
